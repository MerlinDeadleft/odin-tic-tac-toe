# odin-tic-tac-toe

Ninth project (fourth on the node path) of [The Odin Project](https://www.theodinproject.com/) with the goal expanding on object creation with closures and the factory pattern.
[Link to project instructions](https://www.theodinproject.com/lessons/node-path-javascript-tic-tac-toe)

The finished project will show a page where one is able to play a game of tic tac toe.
